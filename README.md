# 🛒 React + Django E-commerce Platform

A full-stack e-commerce web application built using **React.js (Frontend)** and **Django REST Framework (Backend)**.  
This project provides a complete online shopping experience including product browsing, cart management, authentication, and order processing.

---

## 🚀 Features

### 👤 User Features
- User registration & login (JWT Authentication)
- Browse products by categories
- Product search & filtering
- Add to cart & remove from cart
- Place orders & order history
- User profile management

### 🛠 Admin Features
- Add, update, delete products
- Manage categories
- View all orders
- Manage users

### 💻 Other Features
- REST API with Django REST Framework
- Secure authentication (JWT)
- Responsive UI (React)
- State management (Redux / Context API)
- Database integration (PostgreSQL / SQLite)

---

## 🧰 Tech Stack

### Frontend
- React.js
- Axios
- React Router
- Redux / Context API
- Tailwind CSS / Bootstrap

### Backend
- Django
- Django REST Framework
- SimpleJWT (Authentication)
- PostgreSQL / SQLite
